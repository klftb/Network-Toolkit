# 更新日志 (Changelog)

## [v1.0.2] - 2026-06-09

### ✨ 新特性 (New Features)
- **文件传输控制与进度监控**：为 FTP、SFTP 和 TFTP 传输面板全面增加了实时的进度条和传输速度（KB/s、MB/s）显示。
- **动态任务控制**：为 FTP / SFTP 传输新增了基于底层流控拦截的“暂停”与“继续”功能，实现极速毫秒级流控响应；并为所有文件传输（包含 TFTP）增加了“强制终止”按钮，支持一键掐断底层 Socket 网络连接和自动清理临时缓存文件。
- **状态持久化 (Keep-Alive)**：重构了前端路由与界面渲染层逻辑，现在即使您在 FTP/TFTP 服务过程中随意切换到其他网络诊断面板（如 Ping 或 Wi-Fi 扫描），您的文件连接状态和传输任务也绝不会再被重置、中断和丢失。

### 🐛 问题修复与优化 (Bug Fixes & Optimizations)
- **Wi-Fi 信号分布图修复**：重写了 5GHz 频段的信道映射规则（补充了 149 等国内常用信道），并修复了信号衰减绘制距离算法，彻底解决了图表可能空白无信息的问题。
- **测速节点兼容性更新**：南京大学、中科大、东北大学等教育网节点测速网站由于开启了防嵌套（X-Frame-Options）安全策略导致在软件内打不开，现已统一切换为「外部新窗口」安全拉起。
- **FTP 客户端升级**：为 FTP 测试工具新增了 **SFTP 协议** 支持。用户在连接设置中可自由切换协议类型，选择 SFTP 时系统会自动将默认端口变更为 22。

## [v1.0.1] - 2026-06-09

### ✨ 新特性与优化 (Features & Optimizations)
- **UI 视觉升级**：全面优化所有功能卡片和导航图标，调整为 25% 圆角（Rounded-xl）的现代化设计风格，消除过度方正的陈旧感。
- **测速引擎重构**：删除了原本不精确且易卡死的本地模拟测速组件，全新引入「内嵌测速引擎」下拉选项，支持国内教育网/骨干网节点（中国电信、南京大学、中科大、东北大学）及国际开源节点（LibreSpeed、台湾大学）的无缝切换内嵌体验。
- **Wi-Fi 界面优化**：Wi-Fi 扫描列表默认以合并模式（收缩）展示，自动筛选出相同 SSID 下的最强信号，避免大量重名节点刷屏。
- **安装包极致精简**：通过剥离无用的 SourceMap 和剔除多余的 Electron 国际化语言包（仅保留中英双语），将 Windows 打包体积成功压减至 **~94.8 MB**。

### 🐛 问题修复 (Bug Fixes)
- **IP 归属地查询**：修复了原有的 Chinaz 接口被墙/混淆导致数据空白的问题；将功能更名为“IP地址归属”，并全面迁移至 `ip38.com` 引擎；彻底修复了之前国内与国际线路选项无法切换的 Bug。
- **Wi-Fi 测速显示修正**：修复了原先所有无线网统一错误显示为“54Mbit/s”的 Bug。现在系统底层会通过解析真实的无线电类型（Radio Type, 如 802.11ax/ac/n），为您准确映射出现实对应的最大理论带宽。
- **Wi-Fi 信号图表白屏修复**：针对部分 Windows 驱动不返回信道导致坐标轴找不到挂载点而引起的曲线塌陷白屏问题，引入了高容错机制。现已能通过无线频率（如 2412MHz）反向推算信道并成功渲染波峰图表。
